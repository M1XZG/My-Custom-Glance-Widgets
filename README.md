# My Custom Glance Widgets

Custom widget components for the [Glance](https://github.com/glanceapp/glance) dashboard application.

This repository contains reusable custom widgets that extend the functionality of Glance. These widgets can be included in your Glance configuration files using the `$include` directive.

## Related Resources

- **[Glance](https://github.com/glanceapp/glance)** - The main Glance dashboard application
- **[Community Widgets](https://github.com/glanceapp/community-widgets)** - Official community widget repository with additional examples and contributions

## Widgets

- [**bambu-labs-status.yml**](widgets/bambu-labs-status.yml) - Displays Bambu Lab service status
- [**claude-status.yml**](widgets/claude-status.yml) - Shows Claude service status with component details
- [**github-status.yml**](widgets/github-status.yml) - Shows GitHub platform status with component details
- [**vrchat-status.yml**](widgets/vrchat-status.yml) - Monitors VRChat service status with component breakdown
